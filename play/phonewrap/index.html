Phone Wrapper

A Visual Emulator

The phone wrapper was originally a design aid I created while working at airG. It started as a wrapper to emulate the small screens found on yesteryear's feature phones. It helped me prototype and test out designs for the mobile web. As I moved into more modern projects, the wrapper also transformed. However, prototyping and testing tools have come a long way since then. Now, you can do so much just using the developer tools found within Google Chrome.

So why create yet another wrapper? Well, this new version is going to be used on my site and primarily for its aesthetics. It will no longer rely on any graphics or images and its appearance can be generated all via CSS — this makes it easy to implement and customize.

Use the picker to change the phone's appearance. Depending on the width on your screen or window, the landscape option may or may not be available. The phone's scale may also change depending on the available width.

If you want your own Phone Wrapper to use for whatever, here's a zip of the source files — it includes the JavaScript, LESS and CSS files. The source files do not contain any of the scaling logic — it was only needed for this demo.
