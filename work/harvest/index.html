Harvest Frenzy

airG

Harvest Frenzy was another one of the experimental web apps. It is match three style game that borrows from Big Barn World, a sister game that lends itself perfectly to the matching genre.

Much like my work on Letter Lab, the design focus was primarily on the game's UI and UX. We utilized the theme right from start, as the player loads the game. Here's a demo of the game's opening sequence, from the initial load screen to the main start screen.

This demo also relies on the experimental WebKit tags and can only be viewed by browsers that support it.
